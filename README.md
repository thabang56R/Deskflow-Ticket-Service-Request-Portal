DeskFlow is a full‑stack IT ticket management system built with **React**, **Node.js/Express**, and **MongoDB**.  
It provides employees with an easy way to submit and track IT support tickets, while admins can manage, prioritize, and resolve them efficiently.

---

## ✨ Features
- **[Employee Dashboard](ca://s?q=Show_Employee_dashboard_code)** – Submit tickets, view history, track status.  
- **[Admin Dashboard](ca://s?q=Show_Admin_dashboard_code)** – Manage all tickets, update statuses, set priorities.  
- **Authentication** – Role‑based access (Employee vs Admin).  
- **Dark/Light Theme** – Toggle UI theme with context.  
- **Static Pages** – [Help](ca://s?q=Show_Help_page_code), [Privacy](ca://s?q=Show_Privacy_page_code), and [Contact](ca://s?q=Show_Contact_page_code).  

---

## 🛠️ Tech Stack
- **Frontend**: React, React Router, Bootstrap  
- **Backend**: Node.js, Express, JWT Authentication  
- **Database**: MongoDB  
- **Deployment**: Render (Frontend as Static Site, Backend as Web Service)
